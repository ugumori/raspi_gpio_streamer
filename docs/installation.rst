============
Installation
============


Stable release
--------------

To install raspi_gpio_streamer, run this command in your terminal:

.. code-block:: console

    $ pip install raspi_gpio_streamer

This is the preferred method to install raspi_gpio_streamer, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for raspi_gpio_streamer can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/ugumori/raspi_gpio_streamer

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/ugumori/raspi_gpio_streamer/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/ugumori/raspi_gpio_streamer
.. _tarball: https://github.com/ugumori/raspi_gpio_streamer/tarball/master
